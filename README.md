# CloudMusic

#### 介绍
基于Vue全家桶而开发的云音乐播放器，前后端分离，前后的数据

#### 软件架构
后端数据来源于网易云音乐接口

##### 使用技术

- vue2.6.11
- vue-router3.1.5
- axios0.19.2
- element-ui2.13.0

#### 使用说明

1.  npm install
2.  npm run serve
